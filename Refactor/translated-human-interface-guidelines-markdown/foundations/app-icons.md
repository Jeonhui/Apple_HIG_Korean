# **App icons**  

If you have a companion iPhone app, you also need to supply your watchOS app icon in the following sizes.

> 만약 당신이 아이폰 앱을 가지고 있다면, 당신은 당신의 시계도 공급해야 한다다음 크기의 OS 앱 아이콘입니다.
>







| @2x (pixels) | @3x (pixels) |

| --- | --- |

| 58x58 | 87x87 |
